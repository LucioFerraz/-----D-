## Resume
Project developed to analyze the impact of Erasmus+ students (so-called Erasmus Generation) in creating a pan-European identity.

## tools
 - Python, Jupyter Notebook, Panda, Numpy, Tableau.

## data source
- Data sets : https://data.europa.eu
